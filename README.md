# Electricity Consumption Forecasting using Transformer Model

This repository demonstrates predicting electricity consumption using a Transformer-based time series model. The project uses historical electricity consumption data to forecast future usage.

## 📂 Repository Contents
- `Transformer_model.ipynb` – Jupyter Notebook with:
  - Data preprocessing and feature engineering
  - Transformer model implementation
  - Model training and evaluation
  - Visualization of predictions vs actual consumption

## 🧠 Model Overview
- **Model Type:** Transformer for time series forecasting
- **Input:** Historical electricity consumption data
- **Output:** Predicted future electricity consumption
- **Highlights:**
  - Handles sequential time-series data
  - Captures temporal dependencies effectively
  - Includes evaluation metrics and plots
